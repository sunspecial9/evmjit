Detailed Description

Host-endian 256-bit integer.

32 bytes of data representing host-endian (that means little-endian almost all the time) 256-bit integer. This applies to the words[] order as well. words[0] contains the 64 lowest precision bits, words[3] constains the 64 highest precision bits.
